## How it works

Write your articles in markdown and drop them in the `writing/` folder. Add an entry
to `manifest.json` with the slug, title, date, and a short description.

The site uses [marked.js](https://marked.js.org/) to render markdown into the same
typography as the rest of the page — Fraunces for body text, JetBrains Mono for code,
all on the bone background with the teal accent.
